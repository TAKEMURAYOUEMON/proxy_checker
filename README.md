# proxy checker

## using:

### CONFIGURATION:
Paste the proxies to ```proxy_list.txt```

### INSTALL MODULES:
```
pip install colorama
pip install requests
pip install fake_useragent
```
#### TERMUX:
python proxy_check.py

#### WINDOWS:
Open the start.bat or open the proxy_check.py
